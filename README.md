# Campus Crowd

A Flutter application for university students to find less crowded study spaces, cafeterias, libraries, and other campus areas.

## About the App

Campus Crowd helps university students find less crowded areas on campus. The app displays:

- A list of campus areas (Library, Cafeteria, etc.)
- Current crowd levels (Low, Medium, High)
- Color indicators (🟢 Green, 🟡 Yellow, 🔴 Red)
- Time when the data was last updated

## Features

- Home screen with a list of areas and their crowd status
- Color-coded crowd levels
- Refresh button to simulate real-time data updates
- Filter to show only areas with low crowd levels

## Implementation Details

This demo app uses static data that simulates real-time crowd information. In a production environment, this would be replaced with actual data from sensors or user reports.

## Getting Started

To run this application:

1. Make sure you have Flutter installed on your machine
2. Clone this repository
3. Run `flutter pub get` to install dependencies
4. Run `flutter run` to start the app
